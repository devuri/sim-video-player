## Sim Video Player
YouTube and Vimeo player for WordPress, this plugin will create a beautiful HTML5, YouTube and Vimeo media player for all YouTube and Vimeo Videos


### Prerequisites

Tested up to WordPress: 5.4
```
Requires PHP: 5.6+
Tested up to PHP: 7.4
```
